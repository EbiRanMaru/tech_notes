# 日本語関係


## 単語移動
多くのエディタでは、`Ctrl + →` などで、単語区切りでカーソルを移動させることができるが、
Atomは日本語における単語移動をサポートしていない。

### 方法
多くのエディタでは単語を区切る文字列を設定することができる。
Atomの場合、`設定`→`Non Word Characters` で設定できる。

そこに日本語において単語を区切る（ことの多い）文字を入れてしまう。
私は以下を登録している。

```
/\()"':,.;<>~!@#$%^&*|+=[]{}`?-…、。「」【】『』（）！？　てにをはがのともへでや：
```
これだけですごくちょうどよい。
